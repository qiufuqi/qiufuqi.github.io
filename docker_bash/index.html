容器操作

查询所有容器运行状态

创建容器

docker create
新创建的容器默认处于停止状态，不运行任何程序，需要在其中发起一个进程来启动容器。

启动容器

启动容器（一次性执行）

查看容器ip地址

进入容器

进入容器的容器状态必须是up状态 和shell 是两种运行模式

  • docker run -it会创建前台进程，但是会在输入exit后终止进程。
  • docker attach会通过连接stdin，连接到容器内输入输出流，会在输入exit后终止容器进程。
  • docker exec -it 会连接到容器，可以像sSH一样进入容器内部，进行操作，可以通过exit退出容器，不影响容器运行。 使用run进入，是一次性进入

    永久性进入，用docker exec (退出后，容器仍然会运行)

删除容器

不能删除运行状态的容器，只能-f强制删除，或者先停止再删除
已经退出的容器，可以直接删除

查看docker消耗的资源状态

docker stats
