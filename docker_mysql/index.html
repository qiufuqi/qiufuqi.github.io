创建Mysql实例

-v 主机目录：容器目录
/var/lib/mysql (data目录)
/etc/mysql (配置目录)
/var/log/mysql (这个是日志目录)
–privileged=true 是通过root权限操作

开机自启
–restart=always
权限提升
–privileged=true
容器有root权限
配置用户
-e MYSQL_ROOT_PASSWORD=123456
设置初始化root用户的密码

-d mysql:5.7.38
